CNN——迁移学习模型用于图像识别分类
搭建深度学习服务器器对于模型的开放使用很重要，这可以让我们训练好的模型有更多的受众。
但是网上关于此方面的介绍非常少，大多都含糊其辞，一带而过。所以在搭建的时候，自己也遇到了很多困扰，但几乎没有相关的博客能够解决自己的问题。
首先需要购买服务器。我购买的是阿里云服务器，注意：在安装系统的时候一定要选择x64位机器，不然深度学习的一些包可能不能再平台上使用，最明显的就是tensorflow。
机器购买完之后，我们就可以着手将训练好的模型布到服务器上了。
首先需要安装python，服务器我安装的是Ubuntu系统，Ubuntu系统会自带一个python2.7的环境。但是我在本地训练使用的是3.6，好多方法和属性两个版本的python会有
不同，所以首先下载python3.6，而后安装所需的包。
还有一个更为简单的方法，就是直接安装anaconda环境，很多包可以使用conda直接进行安装。
环境准备好之后>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>
将模型上传，在线上可以运行脚本测试一下模型。
>>>>>>>>>>>>>>>>>>>>>>>>>
我所做的项目是APP客户端与服务器端进行交互的系统，所以做完上述工作之后，重点考虑的是如何将APP上传的图片，输入到我们的测试脚本。
这里，我使用的方法是编写脚本，每次上传图片时就调用shell脚本，执行一下测试程序。然后将返回结果反馈给APP客户端。
